# 🚀 SmartTask – Laravel 12 Task Manager

SmartTask is a modern Laravel 12 web app for managing personal or team tasks with a clean UI, authentication, and profile management.

---

## ✨ Features

- ✅ Laravel 12 + Breeze (Blade stack)
- ✅ User Registration & Login
- ✅ Task CRUD (Create, Read, Update, Delete)
- ✅ Profile Edit + Account Delete
- ✅ Tailwind CSS UI (fully responsive)
- ✅ MySQL integration
- ✅ Ready for deployment

## 📦 Installation

```bash
git clone https://github.com/shas88/smarttask.git
cd smarttask
composer install
cp .env.example .env
php artisan key:generate
npm install && npm run build
php artisan migrate
php artisan serve
Open http://localhost:8000

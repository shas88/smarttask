# SmartTask – Laravel Task Manager App

SmartTask is a modern task management web app built with Laravel 12, Tailwind CSS, and MySQL.  
It allows users to register, manage tasks, and update profiles with a clean dashboard experience.

---

### 🌐 Live Demo  
👉 https://smart-taskapp.com

### 🏢 Built by [Sarsha Technologies](https://www.linkedin.com/company/sarsha-technologies/)

---

### 🧪 Features
- User authentication (Laravel Breeze)
- Task creation, editing, and deletion
- Responsive dashboard with Tailwind CSS
- Profile update functionality
- MySQL database
- Deployed on shared hosting with FTP

---

### 🛠️ Tech Stack
Laravel · TailwindCSS · MySQL · PHP · Blade · Git · FTP


---

### 🚀 Getting Started (for developers)
1. Clone the repo  
2. `composer install`  
3. `cp .env.example .env`  
4. `php artisan key:generate`  
5. Set up your DB and run `php artisan migrate`


# 🚀 SmartTask – Laravel 12 Task Manager

## 📦 Installation

```bash
git clone https://github.com/shas88/smarttask.git
cd smarttask
composer install
cp .env.example .env
php artisan key:generate
npm install && npm run build
php artisan migrate
php artisan serve
Open http://localhost:8000
